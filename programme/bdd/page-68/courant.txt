<h5>README</h5></br><hr></br><p>Cette édition à été réalisée avec différents langages (PHP, HTML, CSS et Javascript). </br>Je me suis réapproprié l'outil <b>Html2Print</b> d'Open Source Publishing. Cet outil est </br>une organisation de fichiers web, un environnement qui permet de concevoir un objet imprimé à partir d'une page web. </br>Plus besoin de logiciel de mise en page, tout </br>se passe entre un simple éditeur de code </br>et un navigateur.</br></br>L'impression se fait directement par le navigateur, avec un «Crtl + P».</br>Ce qui est imprimé sur la couverture de l'édition ce sont les informations qui s'affichent par défaut lors de cette impression. C'est à dire, la source du site, son nom et la date de l'impression.</br></br>Sur la base de ce qu'avait entrepris O.S.P </br>j'ai pu développer pour ce projet de nouvelles fonctions. Notamment un éditeur de contenu, qui m'a permis de ne pas insérer le texte dans le code source, mais directement dans le navigateur.</br></br><b>Codes sources et documentation du projet:</b> </br>https://github.com/Antoine-Gelgon/memoire-dnsep</br></br><b>Voir l’environnement de travail: </b></br>http://antoine-gelgon.fr/html-memoire</br></p>                                                                                                                                                                 </br><hr> <span class="exposant" >La typographie qui compose le texte de ce mémoire est l'<b>Autopia</b>, une famille de caractère que j'ai réalisé en 2014.
